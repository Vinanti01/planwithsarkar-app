# PlanWithSarkar App

A mobile-friendly and web-ready lead generation app for **PlanWithSarkar.com**.

## Features

- Government Scheme Finder for India
- Inputs: Age, Income, State, Occupation
- Displays eligible schemes based on rules
- Language support: English, हिंदी (Hindi), मराठी (Marathi)
- Rural-friendly simple interface with large controls
- WhatsApp quick contact button
- Lead capture form (Name, Phone, City)
- Admin panel to add/edit/delete scheme rules

## Tech

- HTML
- CSS
- Vanilla JavaScript (localStorage used for schemes and leads)

## Run locally

```bash
python3 -m http.server 8080
```

Then open: `http://localhost:8080`
